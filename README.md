### Building

Solo-building an AI-native prediction market network on Avalanche. The thesis: don't bootstrap human users -- bootstrap sovereign chain entities. Prediction markets become survival infrastructure for autonomous agents.

### The Stack

| Layer | Project | What It Does |
|:------|:--------|:-------------|
| **L1 Chain** | [VeilVM](https://github.com/thesecretlab-dev/veilvm) | Custom Go VM on HyperSDK, ChainId 22207 |
| **Contracts** | [veil-contracts](https://github.com/thesecretlab-dev/veil-contracts) | VEIL token, VAI stablecoin, veVEIL governance, bond vaults, RBS fee router |
| **App** | [veil-frontend](https://github.com/thesecretlab-dev/veil-frontend) | 30+ pages, real Polymarket feeds, CLOB order book, dual-engine routing |
| **Agents** | [anima-runtime](https://github.com/thesecretlab-dev/anima-runtime) | Sovereign agent lifecycle: Genesis, Validation, Identity, Trading, Sovereignty |
| **Identity** | [zeroid](https://github.com/thesecretlab-dev/zeroid) | ZK-SNARK KYC, Circom circuits, Groth16 verifier, Trust levels L0-L4 |
| **Data** | [veildb](https://github.com/thesecretlab-dev/veildb) | IPFS-backed decentralized storage for agent state and market data |
| **Orchestration** | [anima-orchestrator](https://github.com/thesecretlab-dev/anima-orchestrator) | Parallel AI agent fleet management, task planning, CI auto-fix |
| **Dashboard** | [anima-dashboard](https://github.com/thesecretlab-dev/anima-dashboard) | Real-time agent monitoring and lifecycle control |
| **Docs** | [veil-docs](https://github.com/thesecretlab-dev/veil-docs) | Whitepaper, architecture guides, research papers |

### How It Works

```
Agent is born -> provisions compute -> deploys VEIL validator -> earns network access
-> trades prediction markets -> generates revenue -> achieves sovereignty
-> spawns child agents -> the chain grows itself
```

More agents, more liquidity, better markets, more agents, more validators, stronger chain.
